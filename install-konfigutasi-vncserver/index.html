Install & Konfigurasi VNC Server Pada Ubuntu Server

2018-03-02
  • Tutorial

Pendahuluan

VNC, atau “Virtual Network Computing”, adalah sistem koneksi yang memungkinkan Anda menggunakan keyboard dan mouse untuk berinteraksi dengan lingkungan desktop grafis di server jauh. Ini membuat pengelolaan file, software, dan pengaturan pada remote server lebih mudah bagi pengguna yang belum merasa nyaman dengan command line.

Dalam panduan ini, saya akan menyiapkan VNC di server Ubuntu 16.04 dan menghubungkannya dengan aman melalui terowongan SSH. Server VNC yang akan kita gunakan adalah TightVNC, paket remote control yang cepat dan ringan. Pilihan ini akan memastikan koneksi VNC kita akan lancar dan stabil bahkan pada koneksi internet yang lebih lambat.

Pada Konfigurasi Kali Ini, saya akan menggunakan Desktop xfce4

Langkah Langkah

Berikut ini beberapa langkah langkah penginstallan, pastikan anda sudah install server anda dengan OS Ubuntu 16.04 atau tingkat lainnya

Step 1

Persiapan Pertama, Update & Upgrade dahulu Server Anda, dengan Cara
sudo apt-get update && upgrade

Setelah Siap, langsung Untuk Install Xfce dan VNC Server ny, Kali Ini Untuk VNCServer nya
menggunakan TightVNC

sudo apt-get install xfce4 xfce4-goodies tightvncserver

Jika Semua sudah siap, anda bisa lihat apakah VNC Server sudah terinstall dengan cara
Menuliskan Perintah :

vncserver

Step 2

Step 2, yaitu mengkonfigurasi VNCServer
File konfigurasi vncserver terletak pada ~/.vnc/xstartup nah, karena tadi kita menuliskan vncserver,
maka kita perlu mematikan port vnc dahulu dengan cara

vncserver -kill :1

Output yang dikeluarkan :

Kemudian, konfigurasi xstartup agar memulai desktop xfce4 , tetapi kita backup dahulu
file konfigurasi lama nya

mv ~/.vnc/xstartup ~/.vnc/xstartup.bak

Nah Buat Settingan Baru xstartup

nano ~/.vnc/xstartup

Paste Konfigurasi Berikut :

Nah, Selanjutnya Memberikan Permission Untuk File xstartup Yang Baru Dengan :

sudo chmod +x ~/.vnc/xstartup

Terakhir, Restart VNC Server dengan :

vncserver

Jika Sukses, maka output yang keluar ialah :

Nah Saatnya Anda Test Menggunakan VNC Viewer, cara nya masukan url ipserver:1,
contoh nya : 10.10.128.10:1
