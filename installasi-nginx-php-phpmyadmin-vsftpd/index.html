Pendahuluan

Ini Merupakan Script Bash Untuk Menginstall beberapa Aplikasi Web Server, FTP & Set SSH rc.local dengan mudah, tanpa perlu buka file lainnya, cukup dengan sekali install

OS Server Tested : Saya Menggunakan Debian 9 Stretch & Raspberrypi, Mungkin bisa digunakan untuk Ubuntu 16.04 - 17.10 dan versi tingkat atas nya,

Installasi

  1. instalasi cukup dengan git clone https://github.com/rifkytech/nginx-lemp.git app
  2. Kemudian cd app
  3. setelah itu chmod 755 install
  4. nah selesai, untuk install nya, cukup dengan perintah ./install

Jika Ada Error Perintah, mohon segera hubungi saya@rifky.tech
