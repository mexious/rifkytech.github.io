Bahan Bahan

Untuk bahan” si gk banyak, Cukup Denngan Sebuah Laptop Yang Sudah Diinstal Node.js,
Untuk Yang Belum Silahkan Download Dulu Di Google :D

Step By Step

  • Buka Notepad atau Text Editor Kesukaan anda Isi Dengan script Berikut
  • Save, Beri Nama Sesuka hati, Asalkan Menggunakan Extensi .js
  • Buka Cmd, Sesuaikan Dengan File Direktori Dimana Kamu Menyimpan File .js Tadi
    Contoh, Saya Simpan Di File C:\Users\ANONYMOUS\Documents\NodeTest\apps.js
    Nah Defautl Di CMD nya File C:\Users\ANONYMOUS, Maka Cara Mengubah nya
    Klik Perintah “cd Documents”, “cd NodeTest” . mari Kita Lihat Gambar nya

  • Kemudian Klik Perintah “node namafile.js”. Proses Berjalan Normal Jika ada bacaan “server ini berjalan pada port 9432”

  • Kemudian Buka Browser Anda, Buka Link http://localhost:9432 Dan Siap…
