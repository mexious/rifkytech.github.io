Mempercantik Tampilan Terminal dengan Oh My Zsh

2018-03-31
  • Tutorial

Pendahuluan

Terminal atau Shell, merupakan sebuah aplikasi yang tidak asing lagi bagi para pengguna Linux. nah dalam default nya
biasanya terminal linux menggunakan shell bash. yah bisa dibilang agak membosansih bagi saya jika itu itu saja tampilannya

Nah kali ini saya akan mempercantik tampilan terminal saya dengan Oh My Zsh + zsh

Langkah - Langkah

Perlu kita ketahui, untuk memulai awal langkah langkah nya yaitu mempunyai koneksi internet.
lanjut langkah pertama

Step 1

Pertama Kita perlu menginstall shell zsh, dengan cara :

Untuk Debian / Ubuntu
sudo apt-get install zsh

Untuk Fedora / CentOS / Redhat
sudo yum install zsh

Untuk Arch Linux / Manjaro
sudo pacman -S zsh

Step 2

Kedua, Kita Perlu Membuat shell zsh menjadi default shell di terminal kita

chsh -s /bin/zsh

Step 3

Ketiga, Mari Kita Install OH My ZSH

sh -c “$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)”

Default theme Oh My Zsh menggunakan robbyrussell, dapat diganti dengan mengubah file ~/.zshrc. Cari baris kode ZSH_THEME dan ganti dengan nama theme yang diinginkan. Lihat nama theme dan screenshot-nya di Github

Screenshot terminal di bawah ini menggunakan theme Agnoster. Baca tutorial theme Agnoster agar bisa tampil dengan baik. Dan beberapa theme yang mirip dengan Agnoster dapat dilihat di sini.
